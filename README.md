# Slovofon / Словофон

Slovofon — кроссплатформенное приложение для поиска, прослушивания и оффлайн-загрузки аудиокниг из нескольких источников.

Русское название приложения: **Словофон**.  
Международное название и техническое имя: **Slovofon**.
